# @hominem/lists-react

React components and hooks for list management.

## Exports

### Components
- `Lists.CRUD.*` - List CRUD components (form, row, edit dialog)
- `Lists.Places.*` - Place management in lists (add place, add to list)
- `Lists.Collaboration.*` - Collaboration components (remove collaborator)

### Hooks
- `useLists` - List CRUD operations

## Dependencies

- `@hominem/ui` - UI primitives
- `@hominem/lists-services` - Backend services
- `@hominem/hono-client` - API client
- `@hominem/hono-rpc` - RPC types
- `@tanstack/react-query` - Data fetching
