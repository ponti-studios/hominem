## 🔄 Migration Pattern Overview

### Before (Custom Framework)
```typescript
export default createCommand({
  name: 'auth login',
  summary: 'Authenticate the CLI',
  flags: z.object({ baseUrl: z.string().default('http://localhost:4040') }),
  outputSchema: z.object({ authenticated: z.literal(true) }),
  async run({ flags, context }) {
    // auth logic
    return { authenticated: true };
  },
});
```

### After (oclif)
```typescript
export default class AuthLogin extends Command {
  static description = 'Authenticate the CLI';
  static summary = 'Authenticate the CLI';
  static override flags = {
    baseUrl: Flags.string({ default: 'http://localhost:4040' }),
  };
  static enableJsonFlag = true;

  async run(): Promise<z.infer<typeof outputSchema>> {
    const { flags } = await this.parse(AuthLogin);
    // auth logic
    return validateWithZod(outputSchema, { authenticated: true });
  }
}
```

## 🚀 How to Complete Migration

### For Each Remaining Command:

1. **Update imports:**
   ```typescript
   // OLD
   import { createCommand } from '../../command-factory';
   import { CliError } from '../../errors';
   
   // NEW
   import { Command, Flags, Args } from '@oclif/core';
   import { validateWithZod } from '@/utils/zod-validation';
   ```

2. **Define output schema:**
   ```typescript
   const outputSchema = z.object({
     // your output fields
   });
   ```

3. **Create class:**
   ```typescript
   export default class MyCommand extends Command {
     static description = '...';
     static summary = '...';
     
     static override flags = {
       flagName: Flags.string({ description: '...' }),
     };
     static override args = {
       argName: Args.string({ required: true }),
     };
     
     static enableJsonFlag = true;
     
     async run(): Promise<z.infer<typeof outputSchema>> {
       const { flags, args } = await this.parse(MyCommand);
       // ... business logic ...
       return validateWithZod(outputSchema, output);
     }
   }
   ```

4. **Error handling changes:**
   ```typescript
   // OLD
   throw new CliError({ code: 'X', category: 'Y', message: '' });
   
   // NEW  
   this.error('message', { exit: 2, code: 'X' });
   ```

5. **Context access changes:**
   ```typescript
   // OLD
   context.outputFormat === 'json'
   context.abortSignal
   
   // NEW
   this.jsonEnabled()
   new AbortController().signal
   ```
