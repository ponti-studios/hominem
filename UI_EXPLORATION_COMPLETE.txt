ARCHITECTURE PATTERNS IDENTIFIED

1. Conditional Exports
   - package.json routes based on platform
   - React Native → .native.tsx files
   - Web → .tsx files
   - Types → shared

2. Single Import Path
   - Same import for both platforms
   - Bundler determines which file loads

3. Design System First
   - Shared tokens ensure consistency
   - Platform-specific styling only

4. Framework Flexibility
   - Web: Radix UI + Tailwind
   - Mobile: React Native + Expo

5. Chat Separation
   - index.ts: Web exports
   - index.mobile.ts: Mobile exports
