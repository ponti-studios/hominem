## Why

The Notes workspace plan describes a substantial product direction, but it currently lives outside OpenSpec. It should be captured as an OpenSpec change so the work is scoped, spec'd, and tracked consistently.

## What Changes

- Add an AI-first workspace surface for Notes that keeps the assistant central while surfacing notes, tasks, goals, and events together.
- Support compact note creation, dense workspace browsing, and AI-assisted note actions.
- Preserve the phased implementation plan in OpenSpec artifacts.

## Capabilities

### New Capabilities

- `notes-workspace`: Provides an AI-first workspace surface for notes and related productivity context.

### Modified Capabilities

None.

## Impact

- Affected code: Notes app routes, hooks, workspace panels, AI actions, and related RPC integrations.
- Affected systems: notes UX, workspace navigation, and note action flows.
