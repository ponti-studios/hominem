# OpenSpec Done Index

`openspec/done` is the completed-work home for this repo.

- Current source of truth: `openspec/done/specs`
- Human-readable completion history: `openspec/done/records`
- Raw historical change folders: `openspec/archive`

Legacy paths such as `openspec/specs` and `openspec/merged` remain as compatibility symlinks for the CLI and older tooling, but humans should use `done` first.
